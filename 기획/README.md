## 현재 프로젝트 방향

- Phase 1은 `판단 없는 사실 중심 뉴스 다이제스트`를 만든다.
- 사용자는 매일 아침 5분 안에 주요 뉴스를 분야별로 훑는다.
- 시스템은 고정 뉴스 소스에서 기사를 수집하고, 중복을 줄인 뒤, AI로 짧게 요약한다.
- 결과물은 우선 마크다운 파일로 저장한다.
- 이후 웹, 앱, 이메일, 노션 등 전달 채널을 확장할 수 있다.
