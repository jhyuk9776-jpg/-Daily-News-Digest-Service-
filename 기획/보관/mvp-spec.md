# MVP 기능 명세

## 1. MVP 목표

명령 한 번 또는 자동 실행 한 번으로 오늘 날짜의 뉴스 다이제스트 마크다운 파일을 생성한다.

예상 결과물:

```text
News/2026-06-29.md
```

## 2. 포함 기능

| 기능 | 설명 |
|---|---|
| 뉴스 수집 | 고정된 RSS 또는 뉴스 API에서 기사 제목, 링크, 발행 시간, 요약문을 가져온다. |
| 분야 분류 | 정치, 경제, 사회, 세계, IT/테크 등 분야별로 정리한다. |
| 중복 제거 | 같은 사건을 다룬 여러 기사는 하나로 묶거나 우선순위가 높은 출처를 사용한다. |
| AI 요약 | 기사별 핵심 내용을 1~3문장으로 사실 중심 요약한다. |
| 출처 링크 | 각 요약 항목에 원문 링크를 제공한다. |
| 마크다운 저장 | 날짜별 파일로 저장한다. |
| 오류 표시 | 일부 소스 실패, 요약 실패, 새 기사 없음 등을 문서에 명확히 표시한다. |

## 3. 제외 기능

| 제외 항목 | 이유 |
|---|---|
| 개인화 추천 | Phase 1에서는 기준을 단순하게 유지한다. |
| 의견/해설 생성 | 프로젝트의 핵심은 사실 중심 다이제스트다. |
| 커뮤니티 반응 분석 | Phase 2에서 불편/만족 신호 분석으로 확장할 수 있다. |
| 웹/앱 UI | 먼저 데이터 파이프라인과 결과물 품질을 검증한다. |
| 유료 뉴스 전문 스크래핑 | 저작권과 접근 권한 이슈를 피한다. |

## 4. 결과물 형식 초안

```markdown
# 데일리 뉴스 다이제스트 - 2026-06-29

## 경제

### 기사 제목
- 요약 1
- 요약 2
- 요약 3
- 출처: [매체명](원문 링크)

## IT/테크

오늘 수집된 주요 기사가 없습니다.
```

## 5. 완료 기준

1. 실제 날짜 기준으로 오늘 또는 전날 주요 기사만 포함된다.
2. 분야별로 최소 1개 이상의 뉴스가 들어간다. 단, 새 기사가 없는 분야는 빈 상태를 명시한다.
3. 모든 항목에 원문 링크가 있다.
4. 요약에 확인되지 않은 추측이 들어가지 않는다.
5. 실행 실패 시 원인을 사람이 이해할 수 있는 문구로 남긴다.
