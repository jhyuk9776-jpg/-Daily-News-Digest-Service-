## 2. Phase 1 기획 방향

Phase 1은 큰 뉴스 플랫폼을 만들기 전에, 매일 실제로 쓸 수 있는 최소 뉴스 다이제스트 생성기를 만든다.

핵심 방향은 다음과 같다.

| 항목 | 방향 |
|---|---|
| 제품 형태 | 로컬 Python 기반 뉴스 다이제스트 생성기 |
| 결과물 | 날짜별 마크다운 파일 |
| 핵심 가치 | 빠른 사실 파악, 중복 감소, 출처 확인 |
| 요약 원칙 | 판단, 논평, 과장 표현을 줄이고 사실 중심으로 정리 |
| 범위 | 수집, 선별, 중복 제거, AI 요약, 마크다운 저장 |
| 제외 | 웹/앱, 개인화, 푸시 알림, AI 해설, 의견 비교 |

## 3. 오늘 추천 결정안

| 항목 | 추천 결정 |
|---|---|
| 분야 | 경제, 사회, 세계, IT/테크 |
| 정치 | Phase 1에서는 제외. 필요하면 나중에 별도 옵션으로 추가 |
| 기사 수 | 분야별 2건, 하루 총 8건 |
| 출처 범위 | RSS를 제공하는 국내 주요 매체와 해외 공신력 매체 |
| 저장 위치 | `News/YYYY-MM-DD.md` |
| 실행 방식 | 처음은 수동 실행, 결과물 품질 확인 후 자동화 |
| 자동화 후보 | 1순위 GitHub Actions, 2순위 macOS launchd |
| AI 모델 | 1차 구현은 OpenAI 저비용 모델 우선 검토 |

이 결정안의 의도는 작게 시작해 매일 읽을 수 있는 결과물을 먼저 확보하는 것이다. 웹/앱과 개인화는 결과물 품질이 확인된 뒤 붙인다.

## 6. AI 요약 비용 추정

아래 계산은 기사 1건당 입력 2,500토큰, 출력 250토큰을 가정한 보수적인 초안이다. 실제 RSS 요약문만 쓰면 더 줄어들 수 있고, 본문 전체를 넣으면 늘어날 수 있다.

참고 요금:

- OpenAI API pricing: https://platform.openai.com/docs/pricing
- Anthropic Claude pricing: https://docs.anthropic.com/en/docs/about-claude/pricing
- GitHub Actions billing: https://docs.github.com/en/billing/concepts/product-billing/github-actions

### 기본 MVP

| 기준 | 값 |
|---|---:|
| 하루 기사 수 | 8건 |
| 하루 입력 토큰 | 약 20,000토큰 |
| 하루 출력 토큰 | 약 2,000토큰 |
| 월 실행 횟수 | 30회 |

2026-06-29 확인 기준 주요 모델별 예상 비용은 다음과 같다.

| 모델 | 입력 단가 | 출력 단가 | 하루 예상 | 월 예상 |
|---|---:|---:|---:|---:|
| OpenAI gpt-5.4-nano | $0.20/100만 토큰 | $1.25/100만 토큰 | 약 $0.0065 | 약 $0.20 |
| OpenAI gpt-5.4-mini | $0.75/100만 토큰 | $4.50/100만 토큰 | 약 $0.024 | 약 $0.72 |
| Claude Haiku 4.5 | $1.00/100만 토큰 | $5.00/100만 토큰 | 약 $0.030 | 약 $0.90 |
| Claude Sonnet 4.6 | $3.00/100만 토큰 | $15.00/100만 토큰 | 약 $0.090 | 약 $2.70 |

기본 MVP는 OpenAI gpt-5.4-mini 또는 Claude Haiku 4.5 수준이면 월 1달러 안팎에서 시작할 가능성이 높다. 개발 중 테스트와 재실행을 포함해도 첫 달은 5~15달러 안에서 관리하는 것을 1차 예산으로 둔다.

### 확장 시나리오

| 기준 | 값 |
|---|---:|
| 하루 기사 수 | 20건 |
| 하루 입력 토큰 | 약 80,000토큰 |
| 하루 출력 토큰 | 약 8,000토큰 |
| 월 실행 횟수 | 30회 |

| 모델 | 하루 예상 | 월 예상 |
|---|---:|---:|
| OpenAI gpt-5.4-nano | 약 $0.026 | 약 $0.78 |
| OpenAI gpt-5.4-mini | 약 $0.096 | 약 $2.88 |
| Claude Haiku 4.5 | 약 $0.120 | 약 $3.60 |
| Claude Sonnet 4.6 | 약 $0.360 | 약 $10.80 |

이 경우에도 요약 모델을 저비용 모델로 고르면 월 수 달러 수준에서 운영 가능하다. 다만 기사 본문 전체 수집, 여러 번의 요약 재시도, 이슈 그룹화, 중요도 평가까지 추가하면 비용은 단계적으로 증가한다.

## 6.1 모델 선택 기준

| 기준 | 추천 |
|---|---|
| 최저 비용 | OpenAI gpt-5.4-nano |
| 비용과 품질 균형 | OpenAI gpt-5.4-mini 또는 Claude Haiku 4.5 |
| 요약 품질 우선 | Claude Sonnet 4.6 |
| Phase 1 추천 | OpenAI gpt-5.4-mini로 시작 후 품질 비교 |

Phase 1은 요약이 짧고 형식이 고정되어 있으므로 초반부터 고성능 모델을 쓸 필요는 낮다. 먼저 저비용 모델로 만들고, 표본 기사 10~20개에서 품질이 부족할 때만 모델을 올린다.

## 7. 비용 관리 원칙

1. 처음에는 분야별 2건만 요약한다.
2. RSS 제목과 요약문으로 먼저 동작시키고, 본문 추출은 필요한 기사에만 적용한다.
3. 같은 기사를 반복 요약하지 않도록 URL 기준 캐시를 둔다.
4. 개발 중에는 `--dry-run` 또는 샘플 기사 모드로 API 호출을 줄인다.
5. 실패 재시도 횟수는 제한한다.
