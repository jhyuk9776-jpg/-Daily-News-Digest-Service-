# 결정 로그

이 문서는 앞으로 프로젝트를 진행하며 확정한 내용을 기록한다.  
아직 정하지 않은 것은 `미정`으로 둔다.

## 1. 핵심 결정 현황

| 항목 | 현재 결정 | 상태 | 메모 |
|---|---|---|---|
| 프로젝트 목표 | 매일 아침 사실 중심 뉴스 다이제스트 생성 | 확정 | Phase 1 기준 |
| 1차 결과물 | 날짜별 마크다운 파일 | 임시 확정 | 웹/앱은 이후 |
| 구현 방식 | 로컬 Python 스크립트 | 임시 확정 | 자동화 방식은 추가 결정 필요 |
| 뉴스 분야 | 경제, 사회, 세계, IT/테크 추천 | 승인 필요 | 정치 제외 추천 |
| 뉴스 출처 | 미정 | 결정 필요 | RSS 가능 여부 확인 필요 |
| 출처 우선순위 | 미정 | 결정 필요 | 중복 기사 처리 기준 |
| 하루 기사 수 | 분야별 2건, 총 8건 추천 | 승인 필요 | 읽는 시간 5분 기준 |
| AI 모델 | OpenAI 저비용 모델 우선 검토 | 승인 필요 | Claude는 품질 비교 후보 |
| 자동 실행 방식 | 수동 실행 후 GitHub Actions 추천 | 승인 필요 | 로컬 안정화 뒤 자동화 |
| 저장 위치 | `News/YYYY-MM-DD.md` 추천 | 승인 필요 | 기존 폴더와 연결 가능 |
| 월 AI 비용 예산 | 첫 달 5~15달러 추천 | 승인 필요 | 개발 테스트 포함 |

## 2. 다음에 바로 정할 질문

1. 국내 뉴스만 볼 것인가, 해외 뉴스도 포함할 것인가?
2. 정치 뉴스를 포함할 것인가, 피로도를 줄이기 위해 제외할 것인가?
3. 매일 몇 개의 기사를 읽으면 부담이 없는가?
4. 결과물을 마크다운 파일로만 볼 것인가, 간단한 웹 화면도 빨리 만들 것인가?
5. 자동화는 내 컴퓨터에서 돌릴 것인가, GitHub Actions에서 돌릴 것인가?
6. 첫 달 AI API 비용 상한을 얼마로 둘 것인가?

## 3. 추천 초기값

아래는 빠르게 MVP를 만들기 위한 추천값이다. 사용자가 승인하면 확정으로 바꾼다.

| 항목 | 추천값 |
|---|---|
| 분야 | 경제, 사회, 세계, IT/테크 |
| 정치 | 초기에는 제외하거나 별도 옵션 처리 |
| 기사 수 | 분야별 2건, 총 8건 |
| 출처 | RSS를 제공하는 국내 주요 매체 + 해외 공신력 매체 |
| 결과물 | `News/YYYY-MM-DD.md` |
| 자동화 | 처음은 수동 실행, 이후 GitHub Actions |
| 비용 상한 | 첫 달 5~15달러, 안정화 후 월 1~5달러 목표 |

2026-06-29 기준 오늘은 구현을 바로 시작하기보다, Phase 1의 범위를 더 선명하게 고정한다.
