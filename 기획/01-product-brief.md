# 제품 기획 요약

## 1. 프로젝트 이름

가칭: 데일리 사실 뉴스 다이제스트

## 2. 한 줄 정의

매일 아침, 신뢰할 수 있는 뉴스 출처에서 실제로 일어난 주요 사건을 모아 판단과 해석을 덜어낸 짧은 사실 요약으로 제공하는 AI 뉴스 브리핑 시스템.

## 3. 사용자의 문제

사용자는 세상의 흐름을 알고 싶지만, 매일 여러 뉴스 사이트를 직접 확인하기에는 시간이 부족하다.  
뉴스 피드에는 의견, 자극적 제목, 중복 기사, 알고리즘 추천이 섞여 있어 사실의 뼈대를 빠르게 파악하기 어렵다.

## 4. 사용자가 원하는 결과

- 매일 5분 안에 오늘 중요한 일이 무엇인지 파악한다.
- 특정 매체의 해석보다 사건 자체를 먼저 본다.
- 기획, 취업, 사업 아이디어 발굴을 위한 입력 루틴을 만든다.
- 필요할 때 원문 링크로 깊게 들어간다.

## 5. 핵심 가치

| 가치 | 설명 |
|---|---|
| 빠름 | 짧은 시간 안에 분야별 핵심 뉴스를 확인한다. |
| 사실 중심 | 평가, 추측, 논평보다 사건, 발언 주체, 수치, 날짜를 우선한다. |
| 반복 가능 | 매일 같은 기준과 형식으로 생성된다. |
| 확장 가능 | 처음은 마크다운 파일로 시작하고, 이후 웹/앱/알림으로 확장한다. |

## 6. 제품 원칙

- 사용자는 뉴스를 많이 읽는 사람이 아니라, 세상을 효율적으로 파악하고 싶은 사람이다.
- 요약은 결론을 대신 내려주지 않는다. 판단할 재료를 정리한다.
- 출처 링크는 반드시 남긴다.
- 요약 품질보다 먼저 중요한 것은 출처, 날짜, 중복 제거의 신뢰성이다.
