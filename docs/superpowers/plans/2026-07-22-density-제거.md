# density 완전 제거 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 매체 객관성 지표를 선택률로 일원화하고 density(매체별·당일) 개념을 코드에서 완전히 제거한다.

**Architecture:** density 누적기(`update_media_scores`)와 density 순위(`compute_ranks`)를 삭제하고, rank-history를 선택률 순위(`compute_selection_ranks`)로 교체. 감점 채점기(`score_article`/penalties.yaml)와 당일 감점 메모는 density와 무관하므로 그대로 둔다.

**Tech Stack:** Python 3.9, unittest/pytest, JSON 파일 저장(`scores/media.json`, `scores/media-rank-history.json`, `scores/articles-*.json`).

## Global Constraints

- observe-only 원칙 유지: 이 변경은 선별 산출물(다이제스트)을 바꾸지 않는다. density는 이미 선별 미사용.
- 감점 채점기(`score_article`, `penalties.yaml`, `title_penalty`, `representative_score`)는 **불변**.
- 당일 감점 메모(`total_points` + 표현별 내역)는 유지. `save_article_report`는 density 필드만 제거.
- 선택률 멱등키는 독립(`selection_dates`, curate 흐름) — 건드리지 않는다.
- 최종 수용 기준: `grep -rn density src/` **0건**(주석 포함).
- 작업 브랜치: `feat/remove-density` (이미 생성·체크아웃됨, 스펙 커밋 `7cbb27f` 위에 쌓는다).

```python
def process_date(store: dict, date: str) -> dict:
    """하루치 raw를 채점해 당일 감점 리포트를 저장한다(날짜 멱등)."""
    if date in store.get("processed_dates", []):
        return store
    articles = dated_articles_for(date)
    flags = outlier_flags(articles)
    records = []
    for a in articles:
        ch = {"title": a.get("title", ""), "lead": a.get("summary", ""), "body": ""}
        r = score_article(ch)
        records.append({
            "source": a.get("source", ""),
            "category": a.get("category", ""),
            "title": a.get("title", ""),
            "link": a.get("link", ""),
            "score": r["score"],
            "points": r["points"],
            "hits": r["hits"],
            "attribution": attribution_count(ch),
            "outlier": bool(flags.get(a.get("link", ""))),
        })
    save_article_report(date, records)
    store.setdefault("processed_dates", []).append(date)
    return store
```
