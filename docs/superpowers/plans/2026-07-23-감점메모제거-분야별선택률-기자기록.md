# 감점메모 제거 + 분야별 선택률 + 기자 선택 기록 — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 브리핑에서 선별재설계와 무관한 감점메모를 코드째 걷어내고, 선택률을 분야별로 쪼개며, 최종 선택 기사의 기자를 관찰용으로 누적한다.

**Architecture:** 세 지표가 모두 `curate.select()`의 `selection_stats` 부산물에서 갈라진다 — 선택률(분야별 포함)은 `objectivity.update_selection_rates`, 기자 선택은 `reporters.record_selection`. 감점메모(`objectivity.process_date` 계열)는 유입 원본 전체를 보는 별개 지표라 제거하고, 유일하게 그 CLI에 얹혀 있던 `update_rank_history` 호출만 curate로 이전한다.

**Tech Stack:** Python 3, unittest(pytest로 구동), GitHub Actions cron.

## Global Constraints

- 소표본 보정 없음 — 선택률은 생짜 `win/appear` 유지(2/2=1.0 의도된 동작).
- 공유 코드 불변: `score_article`·`body_objectivity`·`representative_score`·`title_penalty`·`source_coverage`·`compute_selection_ranks`·`update_selection_rates`·`update_rank_history`(정의)·`load_store`/`save_store`. (낚시 게이트·본문 점수·digest_audit·tie-break이 사용)
- 기자 선택 카운트는 순수 관찰 — 블랙리스트·선별 가중치에 반영하지 않음.
- 테스트 실행: `python3 -m pytest tests/ -q` (프로젝트 루트에서).
- 날짜 멱등 유지: 같은 `(date, link)`·같은 `date`는 이중 누적 금지.

def record_selection(data: dict, source: str, author: str, date: str, link: str) -> dict:
    """최종 대표로 선택된 기사의 기자에 선택 카운트를 더한다(관찰용). (date,link) 멱등.

    스트라이크(record_strike)와 같은 파일·같은 '{매체}::{기자}' 키를 공유하되,
    별도 필드(selected_count/selected_history)만 건드린다. 블랙리스트와 무관.
    """
    key = reporter_key(source, author)
    rec = data.setdefault(
        key, {"sparse_count": 0, "points": 0, "blacklisted": False,
              "last_updated": "", "history": []},
    )
    rec.setdefault("selected_count", 0)         # 기존(스트라이크만 있던) 레코드 호환
    rec.setdefault("selected_history", [])
    if any(h["date"] == date and h["link"] == link for h in rec["selected_history"]):
        return data
    rec["selected_history"].append({"date": date, "link": link})
    rec["selected_count"] += 1
    rec["last_updated"] = date
    return data
```

def update_selection_rates(store: dict, daily_stats: list[dict], date: str) -> dict:
    """교차검증 클러스터의 멤버 등장·대표 승리를 매체별로 누적(날짜 멱등).
    전역 selection_rate = win/appear + 분야별(by_category) 동일 누적. 생짜 비율(소표본 보정 없음).
    단독 클러스터는 호출부에서 제외."""
    if date in store.get("selection_dates", []):
        return store
    media = store.setdefault("media", {})
    for cl in daily_stats:
        cat = cl.get("category", "")
        for src in cl["members"]:
            m = media.setdefault(src, {})
            m["appear_total"] = m.get("appear_total", 0) + 1
            bc = m.setdefault("by_category", {}).setdefault(cat, {})
            bc["appear_total"] = bc.get("appear_total", 0) + 1
        w = media.setdefault(cl["winner"], {})
        w["win_total"] = w.get("win_total", 0) + 1
        wbc = w.setdefault("by_category", {}).setdefault(cat, {})
        wbc["win_total"] = wbc.get("win_total", 0) + 1
    for m in media.values():
        appear = m.get("appear_total", 0)
        m["selection_rate"] = (m.get("win_total", 0) / appear) if appear else None
        for bc in m.get("by_category", {}).values():
            a = bc.get("appear_total", 0)
            bc["selection_rate"] = (bc.get("win_total", 0) / a) if a else None
    store.setdefault("selection_dates", []).append(date)
    return store
```
