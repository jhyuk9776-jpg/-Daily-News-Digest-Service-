## Phase 1 — 대표 채점표

목적: 대표 선정에 쓸 두 칼럼 점수 함수. 근거성은 스펙 확정대로 **문장 커버리지**(길이에 안 휘둘리고 0~1). 총합 = 0.6×객관성 + 0.4×근거성. 라벨로 검증(예시1호 > 감점1호).

**Files:**
- Modify: `src/objectivity.py` (`sentence_coverage`, `representative_score` 추가)
- Test: `tests/test_body_scoring.py`

**Interfaces:**
- Consumes: `curate.evidence_signals`(문장별 재사용), `body_objectivity`.
- Produces:
  - `sentence_coverage(body: str) -> float` — 문장 중 증거 신호(숫자·%·기관·인용·기간) 든 문장 비율(0~1). 문장 분리는 `[.!?。\n]`. 빈 본문 0.0.
  - `representative_score(title: str, body: str) -> dict` → `{"objectivity": float, "coverage": float, "total": float}`. objectivity = `1 - min(points,45)/45`, coverage = `sentence_coverage(body)`, total = `0.6*objectivity + 0.4*coverage`. 대표 = `max(members, key=lambda m: representative_score(m.title, m.body)["total"])`.

```python
_SENT_SPLIT = re.compile(r"[.!?。\n]")


def sentence_coverage(body: str) -> float:
    """본문 문장 중 증거 신호(숫자·%·기관·인용·기간)를 담은 문장 비율(0~1).
    길이가 아니라 문장 단위로 정규화 → 긴 기사에 불리하지 않고 상한 1.0."""
    sents = [s for s in _SENT_SPLIT.split(body) if s.strip()]
    if not sents:
        return 0.0
    return sum(1 for s in sents if evidence_signals(s) > 0) / len(sents)
```

```python
def representative_score(title: str, body: str) -> dict:
    """대표 선정용 결합 점수. 총합 = 0.6*객관성 + 0.4*근거성.
    객관성 = 감점(제목+본문) 반전, 근거성 = 문장 커버리지."""
    points = body_objectivity(body, title)["points"]
    objectivity = 1 - min(points, 45) / 45
    coverage = sentence_coverage(body)
    return {"objectivity": objectivity, "coverage": coverage,
            "total": 0.6 * objectivity + 0.4 * coverage}
```
