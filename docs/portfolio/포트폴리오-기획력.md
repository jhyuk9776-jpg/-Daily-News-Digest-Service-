# 누구나 AI를 다루는 시대, 결과를 가르는 건 기획력이다

> 1인이 사용자 관점으로 기획하고, AI를 지휘해 **매일 아침 실제로 가동되는 서비스**까지 만든 기록
> — AI 뉴스 다이제스트

**이재혁** · 2026-07-10 · [라이브 서비스](https://daily-news-digest-service.vercel.app) · [저장소](https://github.com/jhyuk9776-jpg/-Daily-News-Digest-Service-)

AI로 무언가를 만드는 일은 이제 특별하지 않다. 프롬프트 몇 번이면 그럴듯한 결과물이 나온다.
그래서 오히려 변별력은 다른 곳에서 생긴다 — **무엇을, 왜, 누구를 위해 만들 것인가를 정하는
기획력**이다.

이 포트폴리오는 'AI 뉴스 다이제스트'라는 실제 가동 서비스를 만들며 내가 내린 **6개의 기획
결정**으로 그 기획력을 보여준다. 각 결정은 `사용자 관점의 질문 → 내가 내린 결정 →
포기한 것(트레이드오프) → 그래서 무엇이 달라졌나(증거)` 순서로 정리했다. AI는 이 과정에서
손이 빠른 실행 도구였을 뿐, **방향을 정하고 아닌 것을 걸러낸 것은 사람의 판단**이었다.

결과물은 매일 아침 사실 중심으로 요약된 뉴스가 이메일과 웹으로 발송되는, **지금도 돌아가는
서비스**다.
